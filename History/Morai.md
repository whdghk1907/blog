#### **▎ OpenDRIVE API Module 개발 및 최적화**  
- **프로젝트 개요**: OpenDRIVE 변환 작업을 FastAPI 서버로 전환하여 작업 효율성을 극대화하고 재사용성 증대.  
- **핵심 기여**:
  - Python 기반 변환 코드를 FastAPI 서버에 배포하여 변환 작업 중 다른 작업 수행 가능하도록 개선.
  - 대규모 변환 작업 병렬 처리를 위한 Celery, Redis, Kubernetes 도입.
  - 코드 독립성과 관리 효율성을 위해 Submodule 분리 및 전용 Repository 생성.
- **성과**:
  - 현재 단독 사용자 프로토타입 완성 및 다중 사용자 지원 확장 작업 진행 중.

#### **▎ DevOps 및 CI/CD 환경 구축**  
- **목표**: 반복적인 배포 작업 자동화 및 대규모 API 요청 처리를 최적화.
- **주요 작업**:
  - Docker, Jenkins, GitLab Runner, Kubernetes를 활용하여 자동화된 배포 파이프라인 구성.
  - Kubernetes HPA(Horizontal Pod Autoscaler) 적용으로 CPU 부하에 따른 복사본 자동 배포.
  - 클라우드 환경 배포 시 코드 보호를 위해 Cython으로 프로젝트 코드 암호화.
- **성과**:
  - 배포 작업 시간 70% 이상 절감.
  - Celery Queue를 통해 3~10분 소요 작업을 효율적으로 분산 처리.
