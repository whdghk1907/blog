
### **1. 지원 동기**  
자율주행 및 스마트 모빌리티 산업이 빠르게 발전하는 가운데, **정밀도로지도(HD Map) 및 공간정보 데이터 기반의 서비스**는 자율주행, 로봇, AI 서비스의 핵심 요소로 자리 잡고 있습니다.  

그중에서도 **아이나비시스템즈는 내비게이션, 자율주행 통합맵 기술(ADASIS v3 기반)과 퍼스널 모빌리티(스윙 등 전동킥보드/자전거 서비스) 전용 경로 탐색 솔루션 등 다양한 모빌리티 솔루션을 개발하며,  국내 공간정보 서비스 및 모빌리티 기술을 선도하는 기업**입니다.  

저는 공간정보 데이터 처리 및 API 개발을 전문으로 하는 백엔드 개발자로서,  
이러한 혁신적인 환경에서 **클라우드 기반의 확장성 높은 백엔드 시스템을 구축하고,  
자율주행 및 커넥티드 서비스의 핵심 API 개발을 주도하는 역할**을 맡고 싶습니다.  

MORAI에서 **정밀도로지도(HD Map) 및 OpenDRIVE API 개발**을 담당하며,  
**FastAPI 기반 서버 아키텍처를 설계하고 Kubernetes를 활용한 클라우드 환경에서의 서비스 운영을 경험**했습니다.

또한, 지오투 정보기술에서는 **Spring Boot 기반의 GIS 서비스 개발을 수행하며, OpenLayers 및 Geoserver를 활용한 공간 데이터 시각화 프로젝트를 진행**하였습니다.  
이 과정에서 **공간정보 데이터 처리 및 GIS API 개발을 경험하며, 대규모 공간 데이터의 효율적인 관리와 분석을 위한 최적화 작업을 수행**했습니다.

이러한 경험을 바탕으로, **아이나비시스템즈의 모빌리티 프로덕트와 연동하는 REST API 개발 및 클라우드 기반 백엔드 설계**에 기여할 수 있습니다.

### **3. 협업 능력 및 문제 해결 경험**  

#### **1) 피드백을 수용하고 유연하게 협업하는 능력**  
MORAI에서 **자율주행 개발팀과 협업하며** HDMap 데이터를 다루는 과정에서,  
각 팀의 요구사항이 다르고, 지도 데이터를 활용하는 방식에도 차이가 있었습니다.  
이러한 환경에서 **그룹 간 커뮤니케이션을 원활하게 하기 위해, 개발자뿐만 아니라 지도 데이터 제작인원, PM/PL들과의 논의 과정**을 적극적으로 주도했습니다.  

예를 들어, **새로운 Object사이의 WayPoint 충돌 여부 계산 방식과 Belt 아키텍처를 제안하고 이를 API 설계에 반영**한 경험이 있습니다.  
이를 통해 기존보다 **데이터 처리 속도를 50분에서 5분으로 단축**하며 성능을 대폭 개선할 수 있었고,  
연구팀과 개발팀 간 원활한 협업 프로세스를 구축하여 효율적인 데이터 변환 시스템을 완성할 수 있었습니다.  
이처럼 저는 **다양한 의견을 수용하고, 협업을 통해 최적의 해결책을 도출하는 것**을 중요하게 생각합니다.  

#### **2) 문제 해결을 위한 끊임없는 학습 태도**  
공간정보 기술은 빠르게 발전하는 분야이며, 새로운 기술을 학습하고 적용하는 것이 필수적입니다.  
저는 새로운 기술을 배우고 이를 업무에 적용하는 것을 즐기며, 업무에서 발생하는 문제를 적극적으로 해결해 나가는 성향을 가지고 있습니다.  

예를 들어, **GeoDjango 및 PostGIS를 처음 도입할 당시, 기존에 경험이 없던 기술이었지만 공식 문서 및 논문을 참고하며 학습하였고, 이를 활용하여 GIS API 서버를 성공적으로 구축**했습니다.  
이 경험을 통해 새로운 기술을 빠르게 습득하고, 실무에 적용할 수 있는 역량을 기를 수 있었습니다.  
#### **3) 논리적 사고와 명확한 의사 결정**  
API 서버 설계 및 DevOps 환경 구축 과정에서는 **서비스의 안정성과 성능을 고려한 논리적인 의사 결정이 필수적**입니다.  

MORAI에서 **Kubernetes 기반의 CI/CD 환경을 구축하면서**, 여러 배포 전략을 비교 분석한 후,  
**Celery + Redis + Kubernetes HPA 기반의 비동기 태스크 처리 및 자동 확장 아키텍처를 채택하였습니다.**  

이를 통해 **대량의 OpenDRIVE 변환 작업을 병렬로 처리하면서도, 시스템 부하에 따라 자동으로 확장 및 축소되는 유연한 배포 환경을 구축**할 수 있었습니다.  

이처럼 **명확한 근거를 바탕으로 문제를 분석하고, 논리적인 결정을 내리는 역량**을 갖추고 있으며,  
이를 기반으로 **효율적이고 안정적인 공간정보 API 시스템을 구축하는 데 기여**할 수 있습니다.
