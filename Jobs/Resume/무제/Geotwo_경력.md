# **1. 프로젝트 개요**  

### **1.1 GIS 교육 및 실습 멘토링**  
- **Geotools, Geoserver 활용 교육**: OpenLayers, Geotools, Geoserver의 구조적 사용법에 대한 사내 교육 세션 진행.  
- **공간 분석 프로젝트 수행**: OpenLayers, Geoserver를 활용한 공간 분석 및 기능성 검증 프로젝트 진행.  
- **PostGIS DB 실습 진행**: 공간 데이터 관리 및 쿼리 작성 실습 수행.  
- **WMS, WFS, WCS, WPS 활용 실습**: GIS 표준 포맷 데이터를 WKT 타입으로 변환하는 실습 진행.  
- **GIS 웹 프로젝트 실습**: PostGIS 및 공간 분석 쿼리를 적용한 GIS 기반 웹 프로젝트 구현.  

### **1.2 GIS SDK 개발 및 유지보수**  
- **GIS SDK 개발**: OpenLayers 및 Geotools 기반 SDK 서버 관리 및 테스트 페이지 개발.  
  - SDK 샘플 페이지 및 관련 Docs 페이지 작성.  
  - 사용자 입력 파라미터 작성 및 전송 결과 검증 기능 개발.  
- **RESTful API 개발 및 검증**: 데이터 전송과 구성에 대한 리뷰 및 리팩토링 수행.  
- **프론트엔드 개발**: React, TypeScript, Recoil, React-query 등을 활용하여 게시판 및 KanbanBoard 시스템 개발.  
- **해외 프로젝트 리팩토링 및 추가 개발**:  
  - Solution을 활용한 기존 코드 재활용 및 패키징 작업 수행.  
  - 내부 및 외부 프로젝트 코드 리뷰 및 리팩토링 진행.  
- **GIS 국내 프로젝트 유지보수**:  
  - 대량의 엘리베이션 요청을 효율적으로 관리하는 기능 개발.  
  - 200GB 이상의 공간 데이터 관리 스크립트 개발 및 최적화.  

### **1.3 대규모 서버 최적화 및 운영**  
- **배포 서버 최적화**: Jenkins 기반 CI/CD 구축을 통한 배포 프로세스 개선.  
- **서버 데이터 구조 최적화**: PostGIS, Oracle 데이터베이스 마이그레이션 및 구조 최적화 진행.  
- **서버 성능 개선**: 대용량 데이터의 효율적 호출 구조 설계 및 서버 부하 감소 최적화 작업 수행.  

### **1.4 DevOps 환경 구축 및 자동화**  
- **Docker 기반 개발 환경 구성**: Jenkins를 활용하여 개발 및 배포 서버 자동화 구축.  
- **Spring Boot 및 배포 환경 최적화**: React 및 Tomcat 환경에서 배포 프로세스 개선.
