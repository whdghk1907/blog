# Notes - 학습 노트

머신러닝 학습 과정에서 정리한 개인적인 노트와 깨달음을 기록하는 공간입니다.

## 📝 노트 유형

### 개념 정리

- 복잡한 개념을 나만의 방식으로 정리
- 이해하기 어려웠던 부분의 해결 과정 기록

### 실습 회고

- 실습 과정에서 겪은 문제와 해결 방법
- 모델 성능 개선 과정 및 결과

### 학습 일지

- 일별/주별 학습 진행 상황
- 어려웠던 점과 극복 방법

## 📂 구조

```
Notes/
├── Daily-Learning/      # 일일 학습 노트
├── Concept-Notes/       # 개념 정리 노트
├── Practice-Reviews/    # 실습 회고 노트
└── Questions/          # 질문 및 답변 모음
```

## 🎯 활용 방법

- **즉시 기록**: 학습 중 떠오른 생각이나 의문점을 바로 기록
- **정기 회고**: 주간/월간 학습 성과 정리
- **지식 연결**: 새로 배운 내용과 기존 지식의 연결점 찾기
- **MLOps 연계**: 향후 MLOps 학습에 도움될 내용 표시
