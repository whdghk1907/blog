# Supervised Learning - 지도 학습

레이블이 있는 데이터를 사용하여 모델을 학습시키는 머신러닝 방법론입니다.

## 📚 주요 개념

### 회귀 (Regression)
- 연속적인 수치 값을 예측하는 문제
- 예시: 주택 가격 예측, 매출 예측

### 분류 (Classification)
- 카테고리나 클래스를 예측하는 문제
- 예시: 이메일 스팸 분류, 이미지 분류
