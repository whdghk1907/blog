# Unsupervised Learning - 비지도 학습

레이블이 없는 데이터에서 패턴과 구조를 찾아내는 머신러닝 방법론입니다.

## 📚 주요 개념

### 군집화 (Clustering)

- 유사한 데이터를 그룹으로 묶는 작업
- 예시: 고객 세분화, 유전자 분석

### 차원 축소 (Dimensionality Reduction)

- 데이터의 차원을 줄이면서 중요한 정보를 유지
- 예시: 데이터 시각화, 노이즈 제거

### 연관 규칙 (Association Rules)

- 데이터 간의 연관성을 찾는 방법
- 예시: 장바구니 분석, 추천 시스템
