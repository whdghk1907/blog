# Projects - 프로젝트

명확한 결과물과 완료 기한이 있는 프로젝트들을 관리하는 영역입니다.

## 📁 구조

### 🤖 MLOpsBasics
머신러닝 운영 기초 학습 프로젝트
- **Fundamentals/**: 기초 개념 및 구현 방법론
  - **1-1/**: 머신러닝 파이프라인 구축 기초
  - **1-2/**: 프로젝트 구현 방법론

### 📚 MLOps-Zoomcamp
MLOps Zoomcamp 강의 실습 프로젝트

### 🗺️ GISuite
GIS 관련 도구 및 시스템 개발 프로젝트

### 🌐 QGIS
QGIS 활용 프로젝트

## 🎯 프로젝트 관리 원칙

### 📋 프로젝트 정의
1. **명확한 목표**: 구체적이고 측정 가능한 결과물
2. **완료 기한**: 명확한 시작일과 완료 예정일
3. **성공 기준**: 프로젝트 완료를 판단하는 기준

### 📊 진행 상태 관리
- 🟢 **진행 중**: 현재 활발히 작업 중인 프로젝트
- 🟡 **일시 중단**: 임시로 중단된 프로젝트
- 🔴 **완료**: 목표 달성으로 완료된 프로젝트
- ⚫ **취소**: 목표 변경으로 취소된 프로젝트

### 📁 프로젝트 구조
```
ProjectName/
├── README.md          # 프로젝트 개요 및 진행 상황
├── docs/              # 문서 및 계획서
├── src/               # 소스 코드 (해당되는 경우)
├── data/              # 데이터 파일 (해당되는 경우)
└── results/           # 결과물 및 산출물
```

## 🔄 프로젝트 생명주기

1. **계획**: 목표 설정, 일정 계획, 리소스 할당
2. **실행**: 실제 작업 수행 및 진행 상황 추적
3. **완료**: 결과물 정리 및 회고
4. **보관**: Archive로 이동 또는 Areas로 지속 관리 전환
