# Cloud & DevOps 스킬 현황

## AWS 클라우드 스킬

### 현재 보유 스킬
- **AWS CLI**: 설치 및 기본 설정 완료
- **EC2**: 인스턴스 생성, 관리, 종료
- **계정 관리**: 리전별 리소스 관리, 비용 최적화
- **보안**: SSH 키페어 관리, IAM 기본 이해

### 개발이 필요한 영역
- **컨테이너 서비스**: ECS, EKS, Fargate
- **서버리스**: Lambda, API Gateway
- **데이터 서비스**: S3, RDS, DynamoDB
- **MLOps 도구**: SageMaker, ECR
- **모니터링**: CloudWatch, CloudTrail
- **네트워킹**: VPC, Load Balancer
