# Resources - 참조 자료

미래에 유용할 수 있는 참조 자료들을 주제별로 체계적으로 정리한 영역입니다.

## 📁 구조

### 🎓 Skills
기술 및 역량 관련 학습 자료
- **MLOps/**: 머신러닝 운영 관련 개념, 패턴, 학습 자료

### 🔍 Search
검색 및 정보 수집 관련 자료

### 🛠️ Tools
도구 및 기술 가이드
- **AWS/**: Amazon Web Services 관련 설정 및 관리 가이드
- **Docker/**: 컨테이너 기술 관련 설정 및 관리 가이드
- **SSH/**: 원격 접속 및 연결 관련 가이드
- **AI-Tools/**: AI 도구 리뷰 및 활용 가이드

## 📚 자료 유형

1. **개념 정리**: 기본 개념과 이론 설명
2. **실습 가이드**: 단계별 실행 방법
3. **참조 문서**: 공식 문서 요약 및 정리
4. **도구 리뷰**: 도구 비교 및 평가
5. **패턴 모음**: 재사용 가능한 구현 패턴

## 🏷️ 분류 기준

- **기술별**: 사용 기술에 따른 분류
- **용도별**: 활용 목적에 따른 분류
- **난이도별**: 초급, 중급, 고급 수준 구분
- **업데이트**: 최신성 유지 필요 여부

## 🔄 관리 방법

1. **정기 검토**: 분기별 내용 검토 및 업데이트
2. **태그 관리**: 검색 용이성을 위한 태그 체계
3. **연관성**: 관련 자료 간 상호 참조 링크
4. **품질 관리**: 정확성 및 유용성 검증
