# Archive - 보관 자료

완료되었거나 더 이상 활용하지 않는 자료들을 체계적으로 보관하는 영역입니다.

## 📁 구조

### 📅 연도별 분류
- **2024/**: 2024년에 완료된 프로젝트 및 자료
- **2025/**: 2025년에 완료된 프로젝트 및 자료

### 🔧 주제별 분류
- **Technical/**: 기술적 이슈 해결 기록
- **Process/**: 프로세스 및 절차 관련 기록

## 📋 보관 기준

1. **시간 기준**: 6개월 이상 참조하지 않은 자료
2. **완료 기준**: 프로젝트가 완료되어 더 이상 업데이트가 필요 없는 자료
3. **참조 기준**: 가끔 참조할 가능성이 있어 완전 삭제하기 어려운 자료

## 🏷️ 파일 명명 규칙

- 연도별 폴더: `제목-YYYY-MM.md` 또는 `제목-YYYY.md`
- 기술 문제: `문제유형-해결방법-YYYY-MM.md`
- 프로세스: `프로세스명-절차-YYYY-MM.md`
